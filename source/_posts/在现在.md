在现在
人们可以简单的从网上购物
可以简单的刷视频
可以简单的查资料
都是拜 linux 所赐

但是
你是如何做到这一切的呢？
那就是
Windows


Windows 1.0 实际上只是 DOS add a gui，但是他却让许多小白开始使用电脑，个人电脑开始普及

Windows 95 我们有了开始菜单，这个设计影响了快30年，甚至！你使用 KDE Plasma，你都能在左下角看到那个开始菜单（不过我用的是 niri）

Windows 2000，好了，NT 技术，出现了
NT，意为 New Technology，新的技术，新的时代
他是目前所有 Windows 的核心，没有 NT 内核就没有现代 Windows

在国内，大部分人都在使用盗版的 Windows，养成了用户粘性，这导致 Windows 的市场份额非常高。
他当年的睁一只眼闭一只眼，其实是在进行一次 20 年的铺垫
但是微软总是喜欢杀死用户喜欢的功能

在 Windows 11，微软改掉了左下角开始菜单图标，改掉了控制面板
大公司总是喜欢教人做事
这是任何公司都会得的病
但是
无论什么时候
只要我们还能见到那4片花瓣
那4个方块
我们熟悉的开始菜单就还在那
我们的回忆就不会跑
当然
只是现在

完
